Get Started

First step is to know which version you want to use. There are 2 ways to start using Bootstrap, download or use starter template.

Download and Install

You can download the file from the getbootstrap.com and make sure to read through the Installation instructions

Starter Template

If you use the starter template, you can directly open a Visual Studio code, copy the starter template into it and you are ready to make your magic in index.html

Layout

One of essential in building a site is to make a layout and design looks good. Bootsrap layout are covering breakpoints, containers, grid, columns, gutters, utilities and z-index.

Breakpoints

One indication of a a good website is responsive. In order to make it responsive, we need to have a different screen size. Normally, it consist 3 different screen size, from mobile, tablet and desktop. In the coding process, the changing from one screen size to the other screen size called "breakpoints".
