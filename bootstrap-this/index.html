First step is to know which version you want to use. There are 2 ways to start using Bootstrap, download or use starter template.

Starter Template

Layout

One of essential in building a site is to make a layout and design looks good. Bootstrap layout are covering breakpoints, containers, grid, columns, gutters, utilities and z-index.

Breakpoints

One indication of a a good website is responsive. In order to make it responsive, we need to have a different screen size. Normally, it consist 3 different screen size, from mobile, tablet and desktop. In the coding process, the changing from one screen size to the other screen size called "breakpoints". In bootstrap, we can directly use the class infix such as sm,md and lg or customized it using Sass file.

Container

The function of container is to contain all the element in the layout together. If you want to use the grid, then you need to apply class container first. In bootstrap there are container, container-fluid and container breakpoint. Container fluid will filled up all from side to side. Container breakpoint is filled up according to the specific breakpoint.

Grid

Grid system is managing the width for each of element in the layout. It based on 12 grid. Grid helps to make a responsive website. There are several classes to use to get the grid system work, such as breakpoints, container, rows, and col. Col or column class is to set the width in the elements. For example col-6, means that from 12 column, you will use 6 coloumn, or half of the page width.

Components

As the name implies, components contain elements that control the website look and display. Bootstrap components library consist of many elements, such as accordion, alerts, badge, breadcrumb, buttons, card, carousel, collapse, dropdown,list group, modal,pagination, popovers, progress, scrollspy, spinners, toasts, tooltips and navbar ready to use.

Buttons

In bootstrap there are button library style to use for several action, including color, sizes and states. The button color provided based on the action, for example red is shows for danger, yellow is for warning, and green is for success.

Navbar

Navbar is the navigation bar and help to set up. Bootstrap had a library of choices for Navbar ready to use. Navbar cosist of the menu, and using list we can style like active or hover states, alignment, dropdownlink, search and also applying the background color.
